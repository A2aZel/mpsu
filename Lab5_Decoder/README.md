# ЛР5. Декодер инструкций

Декодер инструкций однотактного процессора RISC-V (RV32I + Zicsr + `mret`).
Комбинационная схема, преобразующая инструкцию в набор управляющих сигналов.

## Мои файлы (решение)
- `decoder.sv` — декодер (один `always_comb` со вложенными `case` по
  `opcode`/`func3`/`func7`). При нелегальной инструкции запрещаются все записи
  и переходы (`illegal_instr_o = 1`).

## Вспомогательные файлы
- `decoder_pkg.sv` — коды операций, значения `a_sel`/`b_sel`/`wb_sel`, размеры LDST.
- `alu_opcodes_pkg.sv`, `csr_pkg.sv` — пакеты АЛУ и CSR.
- `lab_05.tb_decoder.sv` — тестбенч (сравнивает с эталонным декодером).

## Как загрузить в Vivado
1. Создать пустой RTL-проект.
2. В `Design Sources` добавить: `alu_opcodes_pkg.sv`, `csr_pkg.sv`,
   `decoder_pkg.sv`, `decoder.sv`.
3. В `Simulation Sources` добавить `lab_05.tb_decoder.sv`
   (верхний модуль симуляции — `lab_05_tb_decoder`).
4. Запустить симуляцию. Тест должен завершиться сообщением `Number of errors: 0`.

В ПЛИС эта работа не проверяется.
