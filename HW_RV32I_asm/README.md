# ДЗ. Программа на ассемблере RISC-V (RV32I) — вариант 19

Файл: `hw_variant19.txt` — программа на ассемблере RISC-V.

## Задание
Вариант 19 из ЛР4: подсчитать число вхождений `11` в двоичном представлении
входного значения **без пересечений** и сохранить результат.

## Данные в ОЗУ
- `input`  — слово с исходными данными (в Venus/Ripes адрес `0x10010000`).
- `result` — слово с результатом (адрес `0x10010004`).
- Результат также дублируется в регистре `x10` (a0).

Чтобы проверить на других данных, измените значение `input` в секции `.data`.

## Как запустить
Открыть `hw_variant19.txt` в эмуляторе [Venus](https://venus.cs61c.org/) или
[Ripes](https://github.com/mortbopet/Ripes) (при необходимости переименовать
в `.s`), нажать Assemble и Run. Результат — в ячейке `result` и в регистре `x10`.

## Проверка
Для примера `input = 0x0000DEAD` (двоично `1101 1110 1010 1101`) программа
насчитывает **4** непересекающиеся пары `11`.
