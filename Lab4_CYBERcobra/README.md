# ЛР4. Простейшее программируемое устройство (CYBERcobra)

Учебный процессор `CYBERcobra 3000 Pro 2.1`, собранный из модулей ЛР1–ЛР3.
Дополнительно реализовано индивидуальное задание (вариант 19).

## Мои файлы (решение)
- `CYBERcobra.sv` — процессор (PC, сумматор PC из ЛР1, `imem`, `rf`, `alu`,
  мультиплексоры источника записи и слагаемого PC, логика `we`/перехода).
- `program_var19.txt` — программа варианта 19 в формате конвертера (с комментариями).
- `program.mem` — та же программа в hex (этим файлом инициализируется память инструкций).
- `cyberconvert.py` — скрипт-конвертер `program_var19.txt` → `program.mem`
  (запуск: `python cyberconvert.py program_var19.txt program.mem`).

### Вариант 19
Подсчёт непересекающихся пар `11` в двоичном представлении `sw_i`.
Алгоритм (жадный): смотрим младшие 2 бита; если `== 11` — счётчик++ и сдвиг вправо
на 2, иначе сдвиг на 1; повторяем, пока число не станет нулём. Результат кладётся
в `r10` и выводится на `out_o` бесконечным циклом в конце.

## Вспомогательные файлы (зависимости из ЛР1–ЛР3)
- `fulladder.sv`, `fulladder4.sv`, `fulladder32.sv`, `alu.sv`, `alu_opcodes_pkg.sv`,
  `instr_mem.sv`, `register_file.sv`, `memory_pkg.sv`.
- `lab_04.tb_cybercobra.sv` — тестбенч.
- `board files/` — проверка в ПЛИС: `nexys_cybercobra.sv` + `nexys_a7_100t.xdc`.

## Как загрузить в Vivado
1. Создать пустой RTL-проект.
2. В `Design Sources` добавить все `.sv` (пакеты + модули + `CYBERcobra.sv`)
   и файл `program.mem`.
3. В `Simulation Sources` добавить `lab_04.tb_cybercobra.sv`
   (верхний модуль симуляции — `lab_04_tb_CYBERcobra`).
4. Запустить симуляцию и вынести на временную диаграмму `pc`, `instr`,
   регистры `rf.rf_mem` и `out_o`. Для проверки варианта 19 задайте на входе
   `sw_i` тестбенча нужное значение (по умолчанию `16'b100001000` — пар `11` нет,
   результат `0`).
5. Для платы: `board files/nexys_cybercobra.sv` + `nexys_a7_100t.xdc`,
   верхний модуль синтеза — `nexys_cybercobra`.

> Проверка результата: на `out_o` появится количество пар `11`. Например, для
> `sw_i = 16'b1110` результат `1`, для `sw_i = 16'b110110` результат `2`.
